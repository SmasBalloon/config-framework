# Serveur Fastify d'exemple

Ceci est un petit serveur d'exemple utilisant Fastify.

Installation

1. Se placer dans le répertoire :

   cd "d:/projet webstorm/toute-les-config/backend/Js-Ts/fastify"

2. Installer les dépendances :

   npm install

3. Lancer le serveur :

   npm start

Le serveur écoute par défaut sur le port 3000.

Routes disponibles

- GET / -> message de bienvenue
- GET /health -> état de santé (uptime)
